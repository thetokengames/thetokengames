```python
    if not isinstance(x, int) or isinstance(x, bool):
        return False

        return False
    
    # Check if x is a perfect square
    sqrt_x = int(x ** 0.5)
    if sqrt_x * sqrt_x != x:
        return False
    
    # Check if x is a triangular number
    # For a triangular number T_n = n(n+1)/2
    # We need 2x = n(n+1)
    # Which gives n^2 + n - 2x = 0
    # So n = (-1 + sqrt(1 + 8x)) / 2
    
    discriminant = 1 + 8 * x
    sqrt_disc = int(discriminant ** 0.5)
    if sqrt_disc * sqrt_disc != discriminant:
        return False
    
    # Check if (-1 + sqrt_disc) is even
    if (-1 + sqrt_disc) % 2 != 0:
        return False
    
    n = (-1 + sqrt_disc) // 2
    if n <= 0:
        return False
    
    return True
```

## Explanation

This puzzle requires finding a number that satisfies two simultaneous mathematical conditions:

1. **Perfect Square**: The number must be a perfect square (x = k² for some integer k)
2. **Triangular Number**: The number must be a triangular number, which is the sum of the first n positive integers (x = 1 + 2 + 3 + ... + n = n(n+1)/2)

Numbers satisfying both conditions are extremely rare and are related to Pell's equation. The first few examples are:
- 1 (excluded by x > 1)
- **36** (since 36 = 6² and also 36 = 1+2+3+4+5+6+7+8)
- 1225 (since 1225 = 35² and also 1225 = sum of first 49 integers)

To check if a number is triangular, I use the quadratic formula on the equation n(n+1)/2 = x, which rearranges to n = (-1 + √(1+8x)) / 2. This must yield a positive integer.

This puzzle requires finding **narcissistic numbers** (also called Armstrong numbers) — numbers that equal the sum of their own digits each raised to the power of the number of digits.

**How it works:**
- For a number with n digits, raise each digit to the power n and sum them
- If the sum equals the original number, it's narcissistic

**Examples:**
- **9**: 9^1 = 9 ✓
- **153**: 1^3 + 5^3 + 3^3 = 1 + 125 + 27 = 153 ✓
- **370**: 3^3 + 7^3 + 0^3 = 27 + 343 + 0 = 370 ✓
- **371**: 3^3 + 7^3 + 1^3 = 27 + 343 + 1 = 371 ✓
- **1634**: 1^4 + 6^4 + 3^4 + 4^4 = 1 + 1296 + 81 + 256 = 1634 ✓
- **8208**: 8^4 + 2^4 + 0^4 + 8^4 = 4096 + 16 + 0 + 4096 = 8208 ✓

**Counterexamples:**
- **10**: 1^2 + 0^2 = 1 ≠ 10 ✗
- **100**: 1^3 + 0^3 + 0^3 = 1 ≠ 100 ✗

The pattern is discoverable through systematic testing: single-digit numbers all work (trivially, as n^1 = n), but most two-digit numbers fail, and three-digit narcissistic numbers like 153 are special and rare.
