def mystery(x):
    if type(x) is not int:
        return False
    if x <= 1 or x >= 10**12:
        return False
    D = 73
    diff = x * x - 1
    if diff % D != 0:
        return False
    y2 = diff // D
    y = int(round(y2 ** 0.5))
    # adjust for float imprecision
    while y > 0 and y * y > y2:

Brute force is futile (~10^12 candidates with virtually no hits), but with the right algorithm the answer is reachable. The next solution exceeds 10^12, so the answer in range is unique.
